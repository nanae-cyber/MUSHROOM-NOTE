# お問い合わせフォームの設定手順

## Googleフォームの作成

1. **Googleフォームにアクセス**
   - https://forms.google.com/ にアクセス
   - Googleアカウントでログイン（kyans_com@yahoo.co.jp または別のGoogleアカウント）

2. **新しいフォームを作成**
   - 「空白」を選択して新しいフォームを作成
   - タイトル：「MUSHROOM NOTE お問い合わせ」

3. **質問を追加**
   
   **質問1: お名前**
   - 質問タイプ：記述式
   - 必須：オン
   
   **質問2: メールアドレス**
   - 質問タイプ：記述式
   - 必須：オン
   - 回答の検証：テキスト → メールアドレス
   
   **質問3: お問い合わせ内容**
   - 質問タイプ：段落
   - 必須：オン

4. **メール通知の設定**
   - 右上の「設定」（歯車アイコン）をクリック
   - 「回答」タブを選択
   - 「メールアドレスを収集する」をオン（任意）
   - 「新しい回答についてのメール通知を受け取る」をオン
   
   ※注意：通知は、フォームを作成したGoogleアカウントのメールアドレスに届きます。
   kyans_com@yahoo.co.jp で通知を受け取りたい場合は、そのアカウントでフォームを作成してください。

5. **フォームのURLを取得**
   - 右上の「送信」ボタンをクリック
   - リンクアイコン（🔗）をクリック
   - 「URLを短縮」にチェックを入れる
   - URLをコピー（例：https://forms.gle/xxxxx）

6. **コードに反映**
   - `src/config.ts` ファイルを開く
   - `CONTACT_FORM_URL` の値を、コピーしたURLに置き換える
   
   ```typescript
   export const CONTACT_FORM_URL = "https://forms.gle/xxxxx"; // ← ここに実際のURLを貼り付け
   ```

## 代替方法：メール転送設定

Googleフォームの通知を別のメールアドレス（kyans_com@yahoo.co.jp）に転送したい場合：

1. フォームを作成したGoogleアカウントのGmailにログイン
2. 設定 → すべての設定を表示 → 「メール転送とPOP/IMAP」
3. 「転送先アドレスを追加」をクリック
4. kyans_com@yahoo.co.jp を入力
5. 確認メールが kyans_com@yahoo.co.jp に届くので、確認リンクをクリック
6. フィルタを作成して、Googleフォームからの通知のみを転送

## 確認

設定が完了したら：
1. アプリのフッターにある「📧 お問い合わせ」リンクをクリック
2. Googleフォームが開くことを確認
3. テスト送信を行い、メールが届くことを確認

## トラブルシューティング

- **メール通知が届かない場合**
  - Googleフォームの設定で「新しい回答についてのメール通知を受け取る」がオンになっているか確認
  - 迷惑メールフォルダを確認
  - Googleアカウントの通知設定を確認

- **フォームが開かない場合**
  - URLが正しくコピーされているか確認
  - フォームの公開設定を確認（「回答を受け付けています」になっているか）
